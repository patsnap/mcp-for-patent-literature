## Quick Start

Install all Life Sciences skills:

```bash
npx skills add patsnap/mcp/tree/main/life-sciences --all
```

Install a specific generated MCP skill directory:

```bash
npx skills add patsnap/mcp/tree/main/ip-intelligence/patsnap-advanced-patent-search
```
